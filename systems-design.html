Systems Design

General Interaction System

Daredevil Fan Game

Tools: UE5, Blueprints, Mixamo

The interaction system is an actor component that can be added to the player to make them interact with certain blueprints. It uses collision boxes to detect when the player is within the range of an interactable object that are instance editable and displays the appropriate UI element. The system supports different types of interactions, such as single press, hold for any duration, and tap repeatedly any number of times.

X-Ray Vision

Daredevil Fan Game

Tools: UE5, Blueprints

X-Ray Vision is a modular, post-processing effect that allows the player to see through walls to locate enemies and items of interest. The effect is achieved using a custom depth pass and a post-process material that highlights objects based on their stencil value. It can be particularly useful when navigating complex environments where visibility is limited, and has different colors for different object types, such as enemies and items, to enhance situational awareness.

Enemy AI

Internship Assignment (work in progress)

Tools: UE5, Blueprints, Mixamo, Fab

This is a comprehensive series of YouTube videos by Ali Elzoheiry on different types of enemy AI, including melee, ranged, and magic enemies. The AI uses behavior trees and blackboards to manage their states and actions, allowing for complex behaviors such as patrolling, chasing, and attacking the player.

This is part of a learning exercise, during my time as a systems designer, to understand AI programming in Unreal Engine 5. The system is designed to be modular and can be easily integrated into any Unreal Engine 5 project. Link to the YouTube series by Ali Elzoheiry.

Parkour System

Daredevil Fan Game (work in progress)

Tools: UE5, Blueprints, Mixamo

Resources: Two YouTube videos (video 1 and video 2) by Matt Aspland that covers a single jump and climbing

The parkour system is a comprehensive movement system that includes various mechanics such as sprinting, mantling, sliding, rolling, grappling hook, and different types of jumps (vault, double jump, triple jump, spin jump, and long jump).

This will be an important part of my Daredevil fan game as Daredevil focuses almost completely on combat and parkour. It is an ongoing project, and I'm learning from a long list of YouTube videos to implement these mechanics using Blueprints in UE5.

After completing this system, I plan to convert it into an actor component that can be easily integrated into any project.
