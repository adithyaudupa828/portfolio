1. ELIXIR BLITZ: SAYA'S TRIALS

Group Project

Roles: UI Design, Level Design, Environment Art, 3D Modeling, Texturing.

In this group project, I created the entire second level. I also created all the UI elements such as the inventory, main menu widget, settings, and the pause menu. I modeled and textured a set of ingredients that the player collects in the game.

Tools: UE5, Blender, Fab, Photoshop, blueprints, widgets, landscape tools, foliage tools.

What I learnt:

Handling both UI and 3D asset modeling for a game was a good challenge. I taught myself how to use Unreal's UMG system to build all the menus and the player's inventory from scratch. For the 3D modeling, I learned how to create low-poly, game-ready ingredients in Blender that would look good without slowing the game down. It was a great lesson in how much these art roles support game design, as the UI had to be clean for the player and the models had to be recognizable at a glance.

2. VILLAGE GAME

Pac-Man Style Game

Roles: UI design, systems design, enemy AI, simple combat.

This is a simple game similar to Pac-Man, but in 3D. The player has to navigate a densely populated environment to collect coins. The villagers have a simple perception system and follow the player while in range. They have a single axe-swing animation that reduces player lives when connected.

I created the combat, player boost, AI behavior, coin collection system, and UI elements. The environment is an asset pack from the Fab store.

Tools: UE5, blueprints, Fab, Mixamo.

What I learnt:

This project was a personal challenge to see if I could build a complete, simple game loop all by myself. I was inspired by Pac-Man and wanted to capture that "collect-and-avoid" tension in a 3D space. The biggest surprise was how tricky it was to design the animations and simple AI for the villagers. It taught me that "simple" is not the same as "easy," and I learned how to connect player controls, AI, UI, and win/lose conditions into one finished package.

3. CASTLE ENVIRONMENT

Roles: Level design, environment art.

Inspired by the YouTube video created by Unreal Sensei, I tried recreating a similar landscape with the same assets. This landscape is for cinematic shots. The castle is made using modular pieces from the Fab, and the landscape is made using Unreal Engine 5's landscape tools and foliage system.

Tools: Landscape tools, foliage system, splines, Fab.

Resources: Assets by Unreal Sensei

What I learnt:

My goal here was to follow a professional tutorial to create a beautiful, cinematic landscape scene. I got completely carried away with the foliage tools and proudly placed over 10,000 trees, bushes, and rocks to make it look dense and realistic. Of course, I had built a scene that was completely unplayable. This taught me about optimization, draw calls, and the critical difference between a "cinematic" and a "game-ready" level.
